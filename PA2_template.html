Consequences of Storm Events

Synopsis

This analysis try to answer two questions:

  1. Across the United States, which types of events (as indicated in the EVTYPE variable) are most harmful with respect to population health?

  2. Across the United States, which types of events have the greatest economic consequences?

After exploring the U.S. National Oceanic and Atmospheric Administration's (NOAA) storm database, it shows that of all severe weather events, HEAT and TORNADO. are the most harmful to population health. But to the greatest economic consequences, HURRICANE and HEAT are the most damage types.

Two columns PROPDMG(property damage) and CROPDMG(crop damage) should take account of their multiplier in PROPDMGEXP and CROPDMGEXP.

multiplier <- function(x, y) {
    x <- as.numeric(x)
    if (y == "K") {
        x <- x * 10^3
    } else if (y == "M") {
        x <- x * 10^6
    } else if (y == "B") {
        x <- x * 10^9
    } else {
        x
    }
}

stormdata1996$PROPDMGRESULT <- mapply(multiplier, stormdata1996$PROPDMG, stormdata1996$PROPDMGEXP)
stormdata1996$CORPDMGRESULT <- mapply(multiplier, stormdata1996$CROPDMG, stormdata1996$CROPDMGEXP)

FATALITIES and INJURIES should be convert to numeric.

stormdata1996$FATALITIESNUM <- as.numeric(stormdata1996$FATALITIES)

Results

Now we can answer the two questions. For the first one, FATALITIES and INJURIES data are aggregated by EVLABEL.

stormdatahealth <- aggregate(cbind(FATALITIESNUM, INJURIESNUM) ~ EVLABEL, data = stormdata1996, 
    sum)
stormdatahealth <- filter(stormdatahealth, (FATALITIESNUM > 0 | INJURIESNUM > 
    0))

Then plot it.

g1 <- ggplot(arrange(stormdatahealth, desc(FATALITIESNUM))[1:10, ], aes(x = reorder(EVLABEL, 
    desc(FATALITIESNUM)), y = FATALITIESNUM)) + geom_bar(fill = "indianred", 
    colour = "black", stat = "identity") + theme(axis.text.x = element_text(angle = 90, 
    hjust = 1)) + xlab("Event Type") + ylab("Fatalities")

g2 <- ggplot(arrange(stormdatahealth, desc(INJURIESNUM))[1:10, ], aes(x = reorder(EVLABEL, 
    desc(INJURIESNUM)), y = INJURIESNUM)) + geom_bar(fill = "steelblue3", colour = "black", 
    stat = "identity") + theme(axis.text.x = element_text(angle = 90, hjust = 1)) + 
    xlab("Event Type") + ylab("Injuries")

grid.arrange(g1, g2, ncol = 2, main = "Fatalities and Injuries by Event Type")

The figure shows that the most harmful events are HEAT and TORNADO.

For the last questions, harmful event to economics is illustrated by same method. HURRICANE is the most damage type for property, and HEAT, for corp damage.
